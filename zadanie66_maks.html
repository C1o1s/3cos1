first line

- obejmuje formatowanie pierwszej lini
p:first-line
{
color:red;
font-size:20pt;
font-weight:bold;
}

after oraz before

- Oba elementy służą do umieszczania tekstu przed i za formatowanym elementem strony
p:before
{
content:"TO SĄ ĆWICZENIA";
font-size:15pt;
font-style:italic;
font-weight:bold;
}
p:after
{
content:"AUTOR Król Lew";
font-size:15pt;
font-style:italic;
font-weight:bold;
}

first-letter

- pozwala na formatowanie pierwszej litery w bloku tekstu.
p:first-letter
{
color:red;
font-size:100pt;
}
